Component Details

ResBlock

Core building block. Processes features + injects time information.

input (C_in × H × W)
↓
GroupNorm → SiLU → Conv 3×3 (C_in → C_out)
↓ + t_emb (MLP: 256 → C_out, broadcast)
GroupNorm → SiLU → Conv 3×3 (C_out → C_out)
↓ + skip (Conv 1×1 if C_in ≠ C_out)
output (C_out × H × W)

Self-Attention

Each pixel attends to all others. Captures global structure.

input (C × H × W) → reshape to (H·W × C)
↓
LayerNorm
↓ split into Q, K, V
Attention = softmax(Q·Kᵀ / √d) · V
↓ + residual
LayerNorm → MLP (Linear → GELU → Linear)
↓ + residual
reshape back to (C × H × W)

Downsample ↓2

Strided convolution — learned spatial compression.

Conv2d(kernel=3, stride=2, padding=1)

Each output pixel sees a 3×3 window of the input, jumping by 2 pixels → halves spatial dimensions.

8×8 → 4×4 (or 16×16 → 8×8)
